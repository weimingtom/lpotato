9.8. 总结

本章，我们讨论了怎么把重复性的命令和循环组合在一起。多数普通的循环使用 for, while 或者 until 语句，或者这些命令的组合来建立。for 循环执行指定次数的循环。 如果你不知道命令应该执行的次数，使用 until 或者 while 来指定循环什么时候应该结束。

循环可以使用 break 和 continue 语句来中断或者继续。

一个文件可以用输入重定向操作符来当作一个循环的输入，循环也可以通过使用管道来读取命令的输出。

select 结构用来在交互脚本中打印菜单。Looping through the command line arguments to a script can be done using the shift statement.
